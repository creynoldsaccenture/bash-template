Sublime Text 3 commands

Setting up user preferences

In Sublime Text 3 you can set various preferences, which makes Sublime Text 3 excellent for code collaboration. For example you can manage coding standards like tabs vs. spaces and so on.

To edit the preferences for your user (the default preferences are read-only in Sublime Text 3), simply go to Preferences > Settings – User.

Code indentation

I would recommend using four spaces for indentation (as this is what most programmers use). You can tell Sublime Text to convert tabs to spaces by opening the User preferences JSON file and changing "translate_tabs_to_spaces" value from false to true, so that you can just press tab to insert four spaces.

Shortcuts

Changing shortcut key bindings

Shortcut key bindings can be changed by editing the JSON file under Preferences > Key bindings – User.

Standard shortcuts

All of the shortcuts listed below are located in the File menu.

Standard Sublime Text 3 shortcuts
Shortcut keys Description
Ctrl + s Save
Ctrl + Shift + s Save as
Ctrl + Shift + n Open new tab
Ctrl + w Close current tab
Ctrl + Shift + t Reopen previously closed tab
Ctrl + Tab Go to next tab
Ctrl + Shift + Tab Go to previous tab
Alt + <n> Go to <n>th Tab (i.e. Alt+2 will switch to the second tab)

It's worth noting that the go to next/previous tab shortcuts navigate in the order in which you last viewed the tabs (rather than in the order they were opened in Sublime Text)
