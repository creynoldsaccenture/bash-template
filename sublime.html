Setting up user preferences

In Sublime Text 3 you can set various preferences, which makes Sublime Text 3 excellent for code collaboration. For example you can manage coding standards like tabs vs. spaces and so on.

To edit the preferences for your user (the default preferences are read-only in Sublime Text 3), simply go to Preferences > Settings – User.
